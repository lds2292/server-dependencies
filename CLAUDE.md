## UI 폰트 색상 기준
다크 테마(배경 `#0f172a`) 기준으로 아래 색상을 사용할 것:
- 주요 텍스트 (제목, 강조): `#f1f5f9`
- 일반 텍스트 (본문, 레이블): `#e2e8f0`
- 보조 텍스트 (설명, 메타정보): `#94a3b8`
- 비활성/힌트 텍스트: `#64748b`
- `#475569` 이하의 색상은 사용하지 말 것 (배경에 묻혀 가독성이 떨어짐)
